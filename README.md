# RN Cooking

A mobile application built with React Native and Expo.

## 🚀 Tech Stack

* React Native
* Expo
* Expo Router
* TypeScript
* Redux Toolkit

## 📦 Installation

Clone the repository:

```bash
git clone <repository-url>
cd rn-cooking
```

Install dependencies:

```bash
npm install
```

## ▶️ Run the project

Start the development server:

```bash
npm start
```

Run on Android:

```bash
npm run android
```

Run on iOS:

```bash
npm run ios
```

Run on Web:

```bash
npm run web
```
